# Garmingps

Garmingps — это веб-приложение на базе Spring Boot, предназначенное для генерации GPX-файлов (GPS Exchange Format) из координатных данных. Позволяет создавать треки и маршруты для использования в GPS-устройствах и приложениях.

## Возможности

- Генерация файлов в формате GPX 1.1
- Поддержка создания треков и маршрутов
- Простой веб-интерфейс
- REST API для программного доступа
- Настраиваемые метаданные треков/маршрутов

## Использованные технологии

- Spring Boot
- Java 21
- Maven 3.6.3

## Быстрый старт

### Сборка приложения

```bash
mvn clean package -DskipTests spring-boot:repackage
```

### Запуск приложения

```bash
mvn spring-boot:run
```

Приложение будет доступно по адресу: `http://localhost:8080`

## API

### Генерация GPX-трека

- **URL**: `/сreate`
- **Метод**: `POST`
- **Content-Type**: `application/json`
- **Тело запроса**:
  ```json
  [
    {"lat": "55.7558", "lon": "37.6173"},
    {"lat": "55.7580", "lon": "37.6215"}
  ]
  ```
- **Ответ**: Скачивание GPX-файла

### Веб-интерфейс

Доступен по адресу `http://localhost:8080`:
- Загрузка координатных данных
- Настройка параметров трека/маршрута
- Скачивание сгенерированных GPX-файлов

## Запуск в Docker

Для развертывания с помощью Docker:

```bash
docker-compose up --build
```
